# Some_Je_Code
Небольшие функцции для удобной работы

Active key in threading - Программа реализует многопоточное отслеживание нажатий клавиш: один поток обрабатывает цифры 0-9, второй — функциональные клавиши s, q, p и ESC для выхода, а главный поток выводит текущее состояние.

Ten views  - Программа реализует выод изображени я камеры. 10 режимов обработки. Сохранение фото по нажитию S начало хзаписи видео по нажатию P и завершение по нажатию Z. Завершение работы программы по Q.
